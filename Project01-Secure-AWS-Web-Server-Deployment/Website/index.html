Secure AWS Web Server Deployment

Amazon EC2 | Apache HTTP Server | Amazon EBS | Red Hat Linux

Project Description

This project demonstrates how to deploy a secure Apache web server on an Amazon EC2 instance running Red Hat Enterprise Linux. Amazon EBS is attached as additional persistent storage, while Security Groups control network access.

Architecture

Internet │ Public IP │ Security Group │ Amazon EC2 │ Apache HTTP Server │ Root EBS │ Amazon EBS (/data)
